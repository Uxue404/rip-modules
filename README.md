## Development
```bash
# Clone the repo
git clone https://github.com/uchue/rip-module

# Install dependencies
bun install

# Run in dev mode
bun run dev
```

## Controls

| Key | Action |
|-----|--------|
| `↑ ↓` | Navigate |
| `space` | Select |
| `enter` | Delete |

## Features

- 🔍 Scans recursively
- 📊 Shows size of each folder
- ✅ Select which ones to delete
- ⚡ Built with Bun + Ink

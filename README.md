# rip-module 🗑️

> Delete all `node_modules` folders fast and easy.

## Install
```bash
npm install -g rip-module
```

## Usage
```bash
# Scan current directory
rip-module

# Scan specific directory
rip-module /path/to/projects
```

## Features

- 🔍 Scans recursively
- 📊 Shows size of each folder
- ✅ Select which ones to delete
- ⚡ Built with Bun
